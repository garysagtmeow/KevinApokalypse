# 🐶 Kevin Apocalypse

## Idee

Kevin Apocalypse ist ein humorvolles 2D-Minigame für das iPhone.

Der Spieler steuert Emi durch ihre Wohnung und versucht, das ständig eskalierende Chaos zu beseitigen.

Kevin verursacht fortlaufend neue Katastrophen, während Timmy überwiegend nutzlos im Sessel sitzt und regelmäßig gerettet werden muss.

---

## Spielprinzip

Der Spieler muss:

- 💩 Hundehaufen einsammeln
- 📦 Kartonschnipsel beseitigen
- 🌱 Pflanzen retten
- 🪣 Erde zusammenkehren
- 💧 Pfützen aufwischen
- 🐶 Kevin gelegentlich einfangen
- 🧔 Timmy retten
- ☣️ verhindern, dass das Mief-O-Meter 100 % erreicht

## Motto

**Kevin trifft keine Schuld.**
